SECOP PRO · Configuración inicial del entorno

Configura tu entorno de análisis de empresas y contratos.

Esta página te guía para validar que Tailwind, el tema claro/oscuro y la base de datos están funcionando correctamente antes de empezar a cargar empresas, contratos y simulaciones.

Paso 1

Verificar estilos

Cambia entre modo claro y oscuro con el botón del navbar. El fondo, textos y tarjetas deben adaptarse suavemente.

Paso 2

Revisar datos de empresas

Accede a la sección de Empresas desde el menú superior para validar la paginación y la tabla.

Paso 3

Probar simulador

Usa el Simulador para generar consorcios y comprobar que las consultas a la base de datos responden bien.

Estado del entorno

Esta sección está pensada para crecer con más checks de salud del sistema: conexiones a base de datos, migraciones pendientes, variables de entorno, etc. Añade aquí cualquier indicador que necesites revisar al iniciar el proyecto.

Base de datos

Asegúrate de que el contenedor de PostgreSQL está levantado y accesible desde la app.

Tailwind

Ejecuta npm run dev:css y comprueba que los cambios de estilo se aplican sin recargar la app.

Modo oscuro

El tema se guarda en localStorage y se respeta la preferencia del sistema.

Datos de ejemplo

Carga algunas empresas y contratos para tener un entorno de pruebas realista desde el primer día.

Guía rápida

Esta página está pensada para superar el alto del viewport y obligarte a hacer scroll, de manera que puedas validar el comportamiento del navbar sticky y el contraste en diferentes secciones.

  • Desplázate hacia abajo para comprobar el sticky del navbar.
  • Alterna entre modo claro/oscuro y revisa textos y bordes.
  • Visita las secciones de Empresas y Simulador desde la navegación.

Próximos pasos recomendados

Utiliza esta sección como checklist vivo para la configuración del proyecto: pipelines de despliegue, backups de base de datos, entornos de staging, etc. Puedes ir marcando tareas aquí o enlazar a herramientas externas de seguimiento.
